  Check out our [walkthrough demonstration](https://youtu.be/sBiP58fYCnw), showcasing the various features that Radiolink provides

<CardGroup cols={2}>
  <Card title="Custom Channels" icon="thought-bubble">
    Access to an unlimited number of **channels**, with custom names, permissions and settings
  </Card>
  <Card title="Unlimited Docks" icon="list-radio">
    Modify and create an unlimited number of **docks**, with permissions for each of them
  </Card>
  <Card title="Permissions" icon="circle-check">
    You can customise who has **access **to docks, including any amount of users and groups
  </Card>
  <Card title="Compatibility" icon="circle-phone">
    Our radios are **compatible **with legagy chat, as well as the most recently new chat service
  </Card>
  <Card title="Battery" icon="battery-half">
    You can toggle **battery** simulation, and how long it takes for the battery to deplete/charge
  </Card>
  <Card title="Portable UI" icon="screencast">
    You can freely transform and **move** the radio interface to where you'd like
  </Card>
  <Card title="Pin UI" icon="map-pin">
    You are able to **pin** the interface, so it doesn't disappear when unequipping the radio tool
  </Card>
  <Card title="Emergency Alerts" icon="siren-on">
    Access to emergency **alerts** and user ability to send and receive a location & description
  </Card>
  <Card title="User Status" icon="calendar-check">
    You can toggle your **status** to Available, Busy or Unavailable using the button provided
  </Card>
  <Card title="Channel Toggling" icon="message-lines">
    Toggle channels with ease to quickly **switch**, whenever you please. Available with <u>B</u> keybind
  </Card>
</CardGroup>
