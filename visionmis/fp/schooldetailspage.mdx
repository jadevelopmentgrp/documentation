---
title: "School Details Page"
description: "School Details Page: displays all information regarding your school, including: name, logo ID, school type, registered years, and group ID"
---

<Warning>
  This is a **non-interactive** page, and does not require interference to manage. It is purely there for realistic purposes, displaying all information for those interested
</Warning>

| **Entity**           | **Description**                                                                                                                                                                                        |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| School Name          | Your school name                                                                                                                                                                                       |
| Short Name           | The shortened version of your school name                                                                                                                                                              |
| Opening dates        | Automatically set to <u>Ongoing</u>                                                                                                                                                                    |
| Logo                 | The decal ID displayed across the system                                                                                                                                                               |
| Establishment number | A number that is automatically set, not yet allocated directly to schools. This is due to be enhanced into a fully-fledged system to where you can search for establishments with Vision in the future |
| School type          | Primary, Secondary, Academy, etc.                                                                                                                                                                      |
| Registered years     | Each year group that is featured as part of logging and management                                                                                                                                     |
| Group ID             | The Roblox group ID for your school                                                                                                                                                                    |

### Session & Vision Details

| **Entity**     | **Description**                                           |
| :------------- | :-------------------------------------------------------- |
| Time/Date      | Displays the current date and time                        |
| Lesson         | Displays the current lesson, if any are ongoing           |
| Vision Version | Displays the current version of Vision you have installed |
| Place ID       | Roblox place ID                                           |
| Session Status | Inactive/Active depending on time                         |
