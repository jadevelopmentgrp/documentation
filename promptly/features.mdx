![Slide1 Jp](/images/Slide1.jpg)

<CardGroup cols={2}>
  <Card title="Vast configuration" icon="list-radio">
    Access to modifying system settings, such as command name, default messages, filter, webhook logging and more
  </Card>
  <Card title="Chat based commands" icon="thought-bubble">
    Built in, modifiable (command name and prefix) chat commands to set and clear messages on the system
  </Card>
  <Card title="Group role access" icon="user-group-simple">
    Access to system settings and message control is dependent on linking the system with a Roblox group ID, and group role IDs
  </Card>
  <Card title="In-game settings" icon="panel-ews">
    Administrators can edit the banner message using chat commands, or using the built in panels for settings
  </Card>
  <Card title="Customisation" icon="sparkles">
    Promptly has various built in customisation settings, such as themes, text colour, visibility, transparency, positioning and plenty more
  </Card>
  <Card title="Scheduled messages" icon="timer">
    Within configuration, you can set up scheduled messages which appear at a specific time of day, depending on the selected timezone
  </Card>
  <Card title="Vision integration" icon="solar-system">
    You can enable Vision integration, and all of your periods will automatically sync to Promptly, and will post a notification and message to users
  </Card>
  <Card title="Cross server integration" icon="server">
    You can send and update settings from one server, and it will sync across all other servers in your game
  </Card>
  <Card title="Smooth interface" icon="joystick">
    We have provided a seamless, enjoyable user experience when interacting with the interface, including delivering smooth animations
  </Card>
  <Card title="Filter configuration" icon="hashtag">
    The configuration has options for filtering text. You can disable filtering, although we are not responsible for any moderation action
  </Card>
  <Card title="Webhook logging" icon="webhook">
    Promptly instantly logs any actions, such as setting messages, or updating settings straight to your Discord webhook
  </Card>
  <Card title="Cooldown" icon="clock">
    Built into the configuration is a configurable cooldown, in seconds, which is the time between waiting to submit another action/message
  </Card>
</CardGroup>
